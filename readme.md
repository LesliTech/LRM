<h3 align="center">Lesli Ruby Message</h3>
<h3 align="center">Message utilities for the Ruby console</h3>

<hr/>

Version 0.1.0


#### Installation
--------
```console
# gemfile
gem "faraday", "~> 0.1.0"

# install
gem install faraday
```


#### Usage
--------
```ruby
LRM.m "Hello ruby world!"
```
